MONTHS 1-9

Foundational Learning

Introduction of Data Science

Duration: 10 clock hours


  • Data science definition and application
  • Sampling and margin of error
  • Descriptive statistics, cross tabulations, and frequencies
  • Data set preparation and data cleansing
  • Regressions analysis
  • SPSS descriptive statistics exercises
  • SPSS regression analysis exercises
  • Pearson correlation, P value
  • Chi Square and ANOVA

Statistics

Duration: 20 clock hours


  • Descriptive statistics fundamentals
  • Inferential statistics fundamentals
  • Confidence intervals
  • Hypothesis testing
  • Simple linear regression
  • Diagnostics and remedial measures
  • Multiple linear regression
  • Modeling and model selection
  • Logistic regression model
  • Linear mixed model
  • Poisson regression model
  • Ordered logic models

SQL

Duration: 14 clock hours


  • Data Definition
  • CREATE, ALTER and DROP Table
  • Data types
  • Default values for columns
  • Primary and Foreign Keys
  • ALTER TABLE to add, remove table columns and constraints etc.
  • Data Manipulation
  • Insert, update, Delete, Drop
  • Data Query
  • SQL Examples
  • Joins, Combining queries, Conditional Expressions, Aggregate function

Power BI

Duration: 22 clock hours


  • Get started with Power BI
  • Connect to external data, clean, and transform
  • Data relationships and calculations
  • Explore your time-based data
  • Data visualizations
  • R integration in Power BI Desktop
  • Create and configure a dashboard
  • Create custom Q&A suggestions
  • Introduction & calculation of DAX

Data Integration

Duration: 15 clock hours


  • Integration data with Excel, SQL
  • Creating Data Warehouse cube
  • Using statistics and DAX formulas
  • Integration of SQL, Data Warehouse into Python

Big Data

Duration: 18 clock hours


  • Introduction to Big Data
  • Big Data Concept
  • Big Data Components
  • Distributed event streaming platforms
  • Distributed file systems
  • HADOOP
  • HDFS Hadoop Distributed File System
  • Map Reduce
  • Hadoop Components Yarn, Hive, Pig
  • Apache Kafka and Apache Pulsar
  • Brokers, Producer - Consumer
  • Topics
  • Connectors
  • Zookeeper

Machine Learning

Duration: 20 clock hours


  • Introduction to machine learning
  • Pandas crash course
  • Data preprocessing
  • Treating missing values
  • Exploratory Data Analysis
  • Supervised Learning overview
  • Logistic regression
  • Linear regression
  • KNN
  • Decision trees
  • Support vector Machines
  • Ensemble Learning
  • Detecting overfitting
  • Oversampling

Python

Duration: 25 clock hours


  • Intro to Python
  • Basic syntax, data types, variable declarations, conditions, loops, strings
  • Data set in Python
  • Working with lists, tuples, and ranges
  • Functions, modules, and object-oriented programming in Python
  • Classes, class instances, objects, constructors
  • Class attributes and methods, access modifiers
  • Getters, setters, and properties
  • Inheritance
  • Introduction to Pandas
  • Working with pandas
  • Working with SQL in python
  • Working with data frames in python
  • File manipulation with python
  • Visualization with python

Personal Branding Course

Duration: 4 clock hours


  • Understanding Personal Branding: Definition and Importance
  • Defining Your Target Audience and Niche
  • Crafting Your Personal Brand Story
  • Content Creation and Curation Strategies
  • Managing Your Online Reputation and Brand Monitoring
  • Leveraging Personal Branding for Career Advancement or Business Growth

GitHub Portfolio Course

Duration: 4 clock hours


  • Introduction to GitHub and Version Control
  • Creating and Customizing Your GitHub Profile
  • Setting Up and Organizing Repositories
  • Working with Branches and Pull Requests
  • Collaborating with Others on GitHub
  • Integrating GitHub with Development Workflows
  • Showcasing Your Projects with GitHub Pages
  • Managing Issues and Enhancing Project Management
  • Implementing Continuous Integration and Deployment (CI/CD) with GitHub Actions
  • Building and Sharing Your Personal Brand on GitHub

Advanced Courses

Performance Management

Duration: 10 Clock Hours

  • Introduction to Performance Management: Definition and Objectives
  • Key Concepts and Components of Performance Management Systems
  • Establishing Key Performance Indicators (KPIs)
  • Performance Planning and Individual Development Plans (IDPs)
  • Monitoring and Measuring Performance: Data Collection and Analysis
  • Providing Feedback and Coaching for Performance Improvement
  • Performance Appraisal Methods and Techniques
  • Addressing Performance Issues and Conducting Performance Reviews
  • Performance Recognition and Rewards
  • Performance Management for Remote Teams or Virtual Work Environments
  • Performance Management Best Practices and Continuous Improvement Strategies

Business Modeling

Duration: 10 Clock Hours

  • Introduction to Business Modeling and its Importance
  • Types of Business Models: Traditional vs. Innovative
  • Business Model Canvas: Understanding the Components
  • Value Proposition Design: Identifying Customer Needs and Solutions
  • Revenue Streams and Pricing Strategies
  • Customer Segmentation and Targeting
  • Channels: Distribution and Delivery Methods
  • Key Resources, Activities, and Partnerships
  • Cost Structure and Financial Modeling
  • Business Model Innovation and Disruption
  • Analyzing and Evaluating Business Models
  • Case Studies of Successful Business Models

DATA + STRATEGY

Duration: 10 Clock Hours

  • Introduction to Data-driven Decision Making
  • Understanding the Role of Data in Strategic Planning
  • Identifying Key Performance Indicators (KPIs)
  • Developing Data-driven Strategies for Business Growth
  • Implementing Data Governance and Quality Management
  • Integrating Data into Strategic Planning Processes
  • Using Predictive Analytics for Strategic Forecasting
  • Data Visualization for Communicating Strategic Insights
  • Case Studies and Practical Applications of Data-driven Strategies

data storytelling

Duration: 4 Clock Hours

  • Introduction and Perception of Visuals
  • Visualizing Data with Graphs
  • Effective Presentation Design
  • Communicating clearly and with vision
  • Building a Persuasive Argument
  • Presenting for Maximum Impact
  • Using visuals to transfer message
